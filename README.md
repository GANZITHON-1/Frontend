- **살펴**

   "우리 동네, 안심하고 **살펴**보다" <br>
   "이웃과 내가 만드는 우리 동네 안전지도, **살펴**"<br>
   
   살펴는 동네의 사건,사고,치안,위험 요소 등을 <br>
   쉽고 직관적으로 파악할 수 있도록 돕는 "**생활 안전 정보 서비스**"입니다.<br>
   
  데이터와 제보를 기반으로, 로컬 이야기를 시각화합니다. <br>
  불편, 민원은 문제 해결과 지역 문화로 연결됩니다.<br>
  주민의 목소리로, 동네를 스스로 지키는 소통 플랫폼입니다.<br>
